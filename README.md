```sh
  $ cd selenium-unittest-is-link-broken-nike/
```

### 3. Install Dependencies
```sh
  /selenium-unittest-is-link-broken-nike$ python -m pip install -r requirements.txt
```

### 4. Execute Program
```sh
  /selenium-unittest-is-link-broken-nike$ python main.py
```
